# Project: Fullstack Todo App

## Purpose of the App

- Hotel page with guestbook
- Guests can leave a message
- Admin area: Guest entries can be deleted and edited

## Built with

- React
- React Router DOM
- TailwindCSS
- RESTful API
- Node.js
- Express.js
- CRUD Routes
- Validation with express-validator
